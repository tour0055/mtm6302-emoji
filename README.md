# Emoji Gallery

This project demonstrates how JavaScript can be used to dynamically create an emoji gallery.

For this assignment, I created an array containing 15 emoji Unicode values. Using a `for` loop, each emoji and its corresponding Unicode code are generated and added to the page through DOM manipulation.

The gallery is displayed using CSS Grid to ensure a responsive layout on different screen sizes. The project is organized using separate HTML, CSS, and JavaScript files.

This assignment helped me practice arrays, loops, DOM manipulation, dynamic content creation, and responsive web design.
